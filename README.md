# Playwright TypeScript POM Framework

## Features
- Playwright with TypeScript
- Page Object Model (POM)
- Fixtures
- JSON Test Data
- API Testing
- HTML Reports
- Screenshot on Failure
- Video Recording on Failure

## Project Structure

Pages/
Fixtures/
tests/
tests-data/

## Run Tests

npm install

npx playwright test

## Generate Report
npx playwright show-report
